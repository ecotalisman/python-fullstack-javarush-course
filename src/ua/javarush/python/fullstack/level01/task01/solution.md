# Preparing to Install Docker on Windows 11

You need to enable the Windows features **Hyper-V** and **WSL 2**. To do this:

1. Open the **Start** menu and search for **Control Panel**.
2. Go to the **Programs** section.
3. Click **Turn Windows features on or off**.
4. Check the boxes for **Hyper-V**, **Virtual Machine Platform**, and **Windows Subsystem for Linux**.
5. Click **OK** and restart your computer to apply the changes.
6. Display the Docker version.

## Requirements:
- Open **Control Panel** on your Windows 11 system.
- Go to the **Programs** section through **Control Panel**.
- Enable **Hyper-V**, **Virtual Machine Platform**, and **Windows Subsystem for Linux** through Control Panel.
- Restart your computer after enabling the required components to apply the changes.
- Display the installed Docker version using the `docker --version` or `docker version` command.

# Підготовка до встановлення Docker на Windows 11

Вам потрібно активувати Windows-сервіси **Hyper-V** та **WSL 2**. Для цього:

1. Відкрийте меню **Пуск** і знайдіть **Панель керування**.
2. Перейдіть у розділ **Програми**.
3. Натисніть **Увімкнення або вимкнення компонентів Windows**.
4. Поставте галочки біля **Hyper-V**, **Virtual Machine Platform** та **Windows Subsystem for Linux**.
5. Натисніть **ОК** та перезавантажте комп’ютер для застосування змін.
6. Відобразіть версію Docker.

## Вимоги:
- Відкрийте **Панель керування** на своїй системі Windows 11.
- Перейдіть у розділ **Програми** через **Панель керування**.
- Активуйте **Hyper-V**, **Virtual Machine Platform** та **Windows Subsystem for Linux** через панель керування.
- Перезавантажте комп’ютер після активації необхідних компонентів для застосування змін.
- Виведіть версію встановленого Docker за допомогою команди `docker --version` або `docker version`.
