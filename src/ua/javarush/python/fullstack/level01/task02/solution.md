# Preparing to Install Docker on Windows 10

You need to enable the **Hyper-V** Windows feature. To do this:

1. Open the **Control Panel**.
2. Go to the **Programs** section.
3. Click **Turn Windows features on or off**.
4. Find **Hyper-V** and check the box next to it.
5. Click **OK** and restart your computer to apply the changes.
6. Display the Docker version.

## Requirements:
- Open **Control Panel** on your Windows 10 system.
- Go to the **Programs** section through **Control Panel**.
- Select **Turn Windows features on or off**, find **Hyper-V**, and check the box next to it.
- Restart your computer after enabling **Hyper-V** to apply the changes.
- Display the installed Docker version using the `docker --version` or `docker version` command.

# Підготовка до встановлення Docker на Windows 10

Вам потрібно активувати Windows-службу **Hyper-V**. Для цього:

1. Відкрийте **Панель керування**.
2. Перейдіть у розділ **Програми**.
3. Натисніть **Увімкнення або вимкнення компонентів Windows**.
4. Знайдіть **Hyper-V** і встановіть галочку поруч із ним.
5. Натисніть **OK** і перезавантажте комп'ютер для застосування змін.
6. Відобразіть версію Docker.

## Вимоги:
- Відкрийте **Панель керування** на своїй системі Windows 10.
- Перейдіть у розділ **Програми** через **Панель керування**.
- Обирайте **Увімкнення або вимкнення компонентів Windows**, знайдіть **Hyper-V** і встановіть галочку поруч із ним.
- Перезавантажте комп'ютер для застосування змін після активації **Hyper-V**.
- Виведіть версію встановленого Docker за допомогою команди `docker --version` або `docker version`.
