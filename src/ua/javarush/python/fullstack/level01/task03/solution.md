# Installing Docker on macOS

You need to install **Docker Desktop** on macOS:

1. Go to the official **Docker** website and download the **Docker Desktop** installer for macOS.
2. Open the downloaded **.dmg** file.
3. Drag **Docker.app** to the **Applications** folder.
4. Open **Docker.app** from the **Applications** folder for the first launch.
5. Enter the administrator password if prompted.
6. **Docker Desktop** will automatically configure and start the **Docker Daemon**.
7. The **Docker** icon will appear in the menu bar, indicating that Docker has started successfully.
8. Display the Docker version using **Terminal**.

## Requirements:
- Go to the official **Docker** website and download the **Docker Desktop** installer for macOS.
- Open the downloaded **.dmg** file and drag **Docker.app** to the **Applications** folder.
- Open **Docker.app** from the **Applications** folder and enter the administrator password if prompted to configure and start the **Docker Daemon**.
- Make sure the **Docker** icon appears in the menu bar, indicating that **Docker Desktop** has started successfully.
- Display the installed Docker version in **Terminal** using the `docker --version` or `docker version` command.

# Встановлення Docker на MacOS

Вам необхідно встановити **Docker Desktop** на MacOS:

1. Перейдіть на офіційний сайт **Docker** і завантажте інсталяційний файл **Docker Desktop** для macOS.
2. Відкрийте завантажений **.dmg** файл.
3. Перетягніть **Docker.app** в папку **Програми**.
4. Відкрийте **Docker.app** з папки **Програми** для першого запуску.
5. Введіть пароль адміністратора, якщо буде запропоновано.
6. **Docker Desktop** автоматично налаштує і запустить **Docker Daemon**.
7. У рядку меню з’явиться значок **Docker**, що вказує на успішний запуск Docker.
8. Показати версію Docker через **Terminal**.

## Вимоги:
- Перейдіть на офіційний сайт **Docker** і завантажте інсталяційний файл **Docker Desktop** для macOS.
- Відкрийте завантажений **.dmg** файл і перетягніть **Docker.app** в папку **Програми**.
- Відкрийте **Docker.app** з папки **Програми** і введіть пароль адміністратора, якщо буде запропоновано, щоб налаштувати і запустити **Docker Daemon**.
- Переконайтесь, що в рядку меню з’явився значок **Docker**, який вказує на успішний запуск **Docker Desktop**.
- Виведіть версію встановленого Docker у **Terminal** за допомогою команди `docker --version` або `docker version`.
