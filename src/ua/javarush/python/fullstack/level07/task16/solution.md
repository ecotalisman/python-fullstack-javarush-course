# Adding a Data Source
# Add Prometheus as a data source in Grafana.
# Go to the settings, select Prometheus, and enter the server URL (http://localhost:9090).
#
# Steps:
#
# 1. Open Grafana at http://localhost:3000.
#
# 2. Log in to the system (admin/admin).
#
# 3. Go to "Configuration" -> "Data Sources".
#
# 4. Click "Add data source".
#
# 5. Select "Prometheus" and enter the URL: http://localhost:9090.
#
# 6. Click "Save & Test".
#
# Requirements:
# • Open Grafana by entering http://localhost:3000 in a web browser.
# • Log in to the system using the admin/admin credentials.
# • Go to the "Configuration" section and select "Data Sources".
# • Click "Add data source", select "Prometheus", and enter the URL: http://localhost:9090.
# • Click "Save & Test" to save and test the connection to the Prometheus data source.
#
