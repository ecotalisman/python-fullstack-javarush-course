## Configuring Histogram Colors

Configure different colors for the bars of successful and unsuccessful requests on the previously created histogram. Change the color of the `statusCode >= 400` metric to red, and `statusCode < 400` to green.

### Steps:

1. Open the histogram on the Dashboard.
2. In the chart settings menu, find the color configuration section.
3. Set the red color for the `statusCode >= 400` filter.
4. Set the green color for the `statusCode < 400` filter.
5. Click **Update** and save the changes.

### Requirements:

1. The histogram must contain red bars for requests with `statusCode >= 400`.
2. The histogram must contain green bars for requests with `statusCode < 400`.
3. Open the histogram on the Dashboard and find the color configuration section to change the color palette.
4. After configuring the colors, click **Update** and save the changes so they take effect.

## Налаштування кольорів гістограми

Налаштуйте різні кольори для стовпців успішних та неуспішних запитів на створеній раніше гістограмі. Змініть колір метрики `statusCode >= 400` на червоний, а `statusCode < 400` — на зелений.

### Кроки:

1. Відкрийте гістограму на Dashboard.
2. У меню налаштування графіка знайдіть розділ із налаштуванням кольорів.
3. Встановіть для фільтра `statusCode >= 400` червоний колір.
4. Встановіть для фільтра `statusCode < 400` зелений колір.
5. Натисніть **Update** і збережіть зміни.

### Вимоги:

1. Гістограма має містити червоний колір для стовпців, які відповідають запитам зі `statusCode >= 400`.
2. Гістограма має містити зелений колір для стовпців, які відповідають запитам зі `statusCode < 400`.
3. Відкрийте гістограму на Dashboard і знайдіть розділ налаштування кольорів для зміни палітри кольорів.
4. Після налаштування кольорів, натисніть **Update** і збережіть зміни, щоб вони набули чинності.
