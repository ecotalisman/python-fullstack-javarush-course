## Incorrect Graphs in Grafana

On the Grafana dashboard, you see incorrect graphs that display incomplete or distorted data. Fix the issue by checking the PromQL queries and graph settings.

### Steps:

1. Check the PromQL queries used in the Grafana panel. Make sure they are built correctly and return the expected data.
2. Configure the graph axes: check the time intervals and labels on the X and Y axes so that the data is displayed correctly.
3. Update the panel and check the visualization settings, such as the graph type, filters, and aggregations.

### Requirements:

1. Make sure that the PromQL queries for the Grafana panels are built correctly and return the expected data according to the visualization goals.
2. Make sure that the time intervals and labels on the X and Y axes are configured so that the data is displayed correctly without distortion or incompleteness.
3. Update the Grafana panel and check the visualization settings, including the graph type, filters, and aggregations, to make sure that the displayed information is correct.

## 🇺🇦 Ukrainian version:

## Некоректні графіки в Grafana

На дашборді Grafana ви бачите некоректні графіки, які відображають неповні або спотворені дані. Виправте проблему, перевіривши запити PromQL та налаштування графіків.

### Кроки:

1. Перевірте запити PromQL, які використовуються в панелі Grafana. Переконайтеся, що вони правильно побудовані та повертають очікувані дані.
2. Налаштуйте осі графіків: перевірте часові інтервали та мітки на осях X і Y, щоб дані коректно відображалися.
3. Оновіть панель і перевірте правильність налаштувань візуалізації, наприклад, тип графіка, фільтри та агрегації.

### Вимоги:

1. Перевірте, щоб запити PromQL для панелей Grafana були правильно побудовані та повертали очікувані дані, відповідні цілям візуалізації.
2. Переконайтесь, що часові інтервали та мітки на осях X і Y налаштовані так, щоб дані коректно відображалися, не викликаючи спотворень або неповноти.
3. Оновіть панель Grafana і перевірте налаштування візуалізації, включаючи тип графіка, фільтри та агрегації, щоб впевнитися в коректності відображуваної інформації.
